# UserInfoCollector - User Information Program

## What it does

Asks for your name, gender, age, place, and a character. Then it prints everything back with a funny message.

## How to run

Compile it:
```bash
javac UserInfoCollector.java
```

Run it:
```bash
java UserInfoCollector
```

## Example

**Input:**
```
Name: Ahmed
Gender: Male
Age: 25
Place: Riyadh
Enter a character: A
```
